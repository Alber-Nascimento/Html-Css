Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântico).

Nesta frase temos um termo em ênfase usando a tag EN (semântico).

Texto marcado

Podemos também criarmos um texto marcado com a tag MARK.

Texto grande / Texto pequeno

Neste parágrafo temos: Texto grande / Texto pequeno

Texto deletado

Podemos marcar um texto como deletado para indicar que o mesmo deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para darmos ênfase e indicar que o mesmo foi inserido depois.

Texto sobrescrito

Podemos escrever coisas do tipo: x20+3.

Texto subscrito

Podemos escrever coisas do tipo: H2O.
